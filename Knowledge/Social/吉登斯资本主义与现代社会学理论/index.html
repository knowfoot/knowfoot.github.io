吉登斯资本主义与现代社会学理论

提到安东尼·吉登斯，首先想到的是现代社会学的态度。他的《资本主义与现代社会学理论》在社会学专业的学生中无人不知，无人不读，仅次于马克思·韦伯的《新教伦理与资本主义精神》。

安东尼·吉登斯是英国著名的社会学家，他提出了“结构化理论”，对现代社会进行了深入的分析和阐释。在著作《资本主义与现代社会学理论》中，吉登斯论述了资本主义的发展及其对现代社会的深刻影响。

资本主义的发展

吉登斯认为，资本主义是一种独特的经济制度，它具有以下几个特征：

  1. 私有制：**资本主义经济中的生产资料归私人所有。
  2. 市场竞争：**资本主义经济中的企业相互竞争，以获取利润。
  3. 利润动机：**资本主义经济中的企业以追求利润为主要目标。
  4. 资本积累：**资本主义经济中的企业不断积累资本，以扩大生产规模。

吉登斯认为，资本主义经历了几个主要的发展阶段：

  1. 商业资本主义：**16世纪至18世纪，资本主义主要以商业贸易的形式存在。
  2. 工业资本主义：**18世纪至19世纪，资本主义主要以工业生产的形式存在。
  3. 金融资本主义：**19世纪末至20世纪初，资本主义主要以金融业为主导。
  4. 全球化资本主义：**20世纪末至今，资本主义以全球化的形式存在。

资本主义对现代社会的影响

吉登斯认为，资本主义的发展对现代社会产生了深刻的影响。这些影响包括：

  1. 经济增长：**资本主义经济的快速增长带来了物质财富的极大丰富。
  2. 社会分化：**资本主义经济的发展导致了社会阶层的分化，出现了资产阶级和无产阶级。
  3. 城市化：**资本主义经济的发展导致了城市人口的快速增长，出现了大城市和城市化地区。
  4. 全球化：**资本主义经济的发展导致了全球经济的相互联系，形成了全球化的世界经济。
  5. 现代性：**资本主义经济的发展带来了现代性的价值观和生活方式，如个人主义、理性主义和科学主义。

吉登斯的结构化理论

吉登斯提出了“结构化理论”来解释资本主义的发展及其对现代社会的影响。结构化理论认为，社会结构既是行动的产物，又是行动的条件。社会结构为行动提供了框架和限制，但行动者也可以通过他们的行动来改变社会结构。

吉登斯认为，资本主义经济是一种结构，它既是行动的产物，又是行动的条件。资本主义经济为人们的经济活动提供了框架和限制，但人们也可以通过他们的经济活动来改变资本主义经济。

  1. 资本主义的经济增长：**资本主义经济的快速增长带来了物质财富的极大丰富。这使得人们的生活水平得到了显著提高，也为社会的发展提供了物质基础。
  2. 资本主义的社会分化：**资本主义经济的发展导致了社会阶层的分化，出现了资产阶级和无产阶级。这导致了社会的矛盾和冲突，也为社会变革提供了动力。
  3. 资本主义的城市化：**资本主义经济的发展导致了城市人口的快速增长，出现了大城市和城市化地区。这带来了许多社会问题，如住房问题、交通问题和环境问题。
  4. 资本主义的全球化：**资本主义经济的发展导致了全球经济的相互联系，形成了全球化的世界经济。这使得世界各国之间的经济联系日益紧密，也为全球合作与发展提供了机遇。

吉登斯的结构化理论与马克思的唯物史观有相似之处，但也有不同之处。马克思认为，经济基础决定上层建筑，经济结构决定了社会结构。吉登斯则认为，社会结构既是行动的产物，又是行动的条件。社会结构为行动提供了框架和限制，但行动者也可以通过他们的行动来改变社会结构。

总结

看看咱们这谁叫牛逼哄哄？英国社会学界自从火起来的马林诺夫斯基开始反思前辈们的行为艺术以后，一晃快半个世纪了，愣是没人能赶得上伟大的韦伯伯爵、帕森斯老爷子或者戈夫曼大哥那种霸气。直到我们的70年代英勇小将吉登斯横空出世，发表那部震撼人心的《资本主义与现代社会理论》，咱们社会学才又找回了霸主地位。这书的副标题，大家可看到了没？《对俺们老马、涂尔干、韦伯三剑客遗作的长途马拉松分析》，这名字看着就让人热血沸腾！不仅如此，吉登斯先生还是把一连串的硬货儿端到了桌子上，整整30多本都是响当当的。就因为他在社会学方面的成就，使得人们往往只把他看做一位杰出的社会学家，却忽视了他其实还是一位走在时代前列的政治大哲！

许多热心研习政治哲学的同学可能会忘了，其实啊，俺们可以从哲学出发去理解政治哲学，也可以从别的那些玩意儿，比如说经济学、社会学、政治学、法学等各行各业里去了解。比如说，哈耶克先生就是从政治经济学转行过来的；吉登斯大哥呢，则是从社会学那儿跳进了政治哲学的大门。

他是从社会学的角度来解读现代性的，大作就是那本鼎鼎大名的现代性的后果。这部作品深度剖析了20世纪让人头疼不已的现代化问题，堪称一本解决现代性危机的百宝箱。他在书中开门见山地表示：“在这个宝典中，我会据交叉对比文化和认识论，从制度层面来解读现代性这位同志。”

他尖锐地指出，现代社会制度在组织架构上可谓是独步天下，理论上要胜过所有的传统秩序不得了 。他认为过去的那些社会图谱如今已悄然变革，但对于新形态却还没有合适的趋势分析。像马克思、涂尔干、韦伯他们这些大神，在分析社会问题时惯用一招数，试图以单一的驱动力驾驭社会这个庞然大物。马克思偏爱如同烈日炎炎般的资本主义，涂尔干则痴迷工业革命的诱惑，而韦伯则对科层制社会客体组织情有独钟。

吉登斯先生向我们揭示了现代性所面临的那个一言难尽的困境：突飞猛涨的人口、经济机制的轰然崩塌、生态危机的愈演愈烈、核战争的威胁以及规模浩大的战争，这些都是摆在人类面前的难题。他提出了解决之道。吉登斯的社会学理论那可是相当丰富，在处理现代性危机这种问题时，甚至比哈贝马斯这些家伙还要有说服力。

他在那本书中，《第三条道路——社会民主主义复兴》里提出了他自己独特的政治哲学理念，翻译成中文便是《第三条道路》。这名字挺有意思吧，就意味着既不是和恁么高大上的右派老哈耶克唱对台戏（老哈耶克那一套叫做自由至上主义），也不是和老甘苦大众站一起的左派马克思（那是老左派的社会民主主义），而是那种走在中间的，积极向上的社会民主主义。

他主张公平公正、关爱弱势群体、有限度地进行基因调整，重新激活公共空间、大力推进福利政策、建设全民智慧型国家，超越劳动为主导的社会模式等等一堆政治哲学观点。

他特别关心咱们西方社会两极分化问题严重的现状，可以说，吉登斯的这本《第三条道路》本质上还是偏向社会主流的，群众路线，实实在在的发展道路。他深感马克思关于无产阶级思想的号召力。

这吉登斯可真是毫不吝啬地表露了自己深深深受马克思主义的熏陶啊！来看他那套正儿八经的说法吧，人、社会、国家这仨货在他眼里头，可是人跟社会的关系要比跟国家的关系重要得多啦。他说，要是政府啥都不干，给老百姓们弄点实际好处，那它就是失去了行使公权力的资格；而那些不愿意积极参与改善自己生活环境的人们呢，也就别想得到平等的掩护了。

这个吉登斯还特别提到了这么一茬子事儿:“社会吸纳”中的那个“劳动”被单一化了，这就让社会与人们之间的关系只剩下劳动关系了。大家都知道，这可不是啥好事儿。

这么说吧，吉登斯这家伙最关心的问题就是人跟社会这个大组合体之间的关系啦。他引用马克思的那句名言说，马克爷爷认为嘛，等到社会主义社会高度发达起来之后，我们这个大号的国家就会消失不见，然后在这个大家共享的社会中，每位公民的自由发展都将成为其他所有每个人自由发展的前提。吉登斯先生深信，如今的人类不仅仅只是受到国家的困厄，更是被社会无视冷落得不得不互相告别传统的人际关系，这个现代性的危机使得后现代的生活境况越发严峻，导致整个社会的共同价值观和个人自由变得空洞无物、纸上谈兵。

大家都说，吉登斯这家伙的政治哲学思想里面融合了马克思的社会主义思想，桑德尔的共享主义，阿伦特的个人政治自由和公共声誉等各种元素，同时也毫不留情地抨击了新古典自由主义所带来的弊端，指出自由主义在推动现代化进程的同时，也酿成了现代性困境，拖累了整个社会让它四分五裂。

总之呢，吉登斯这老大哥可以说是20世纪思想界的一个里程碑式的人物。他不光是英国政治经济学院院长，还是一位终生担任教授的厉害角色，而且还是当时的英国首相布莱尔的脑袋智囊团成员。他无论是在学术领域还是在现实政治舞台上都有着无比巨大的影响力。对于他同时代的哈贝马斯这帮人的观点，他也不下心说了几句怪话批评一番。
