吉登斯资本主义与现代社会学理论

提到安东尼·吉登斯，首先想到的是现代社会学的态度。他的《资本主义与现代社会学理论》在社会学专业的学生中无人不知，无人不读，仅次于马克思·韦伯的《新教伦理与资本主义精神》。

安东尼·吉登斯是英国著名的社会学家，他提出了“结构化理论”，对现代社会进行了深入的分析和阐释。在著作《资本主义与现代社会学理论》中，吉登斯论述了资本主义的发展及其对现代社会的深刻影响。

资本主义的发展

吉登斯认为，资本主义是一种独特的经济制度，它具有以下几个特征：

  1. 私有制：**资本主义经济中的生产资料归私人所有。
  2. 市场竞争：**资本主义经济中的企业相互竞争，以获取利润。
  3. 利润动机：**资本主义经济中的企业以追求利润为主要目标。
  4. 资本积累：**资本主义经济中的企业不断积累资本，以扩大生产规模。

吉登斯认为，资本主义经历了几个主要的发展阶段：

  1. 商业资本主义：**16世纪至18世纪，资本主义主要以商业贸易的形式存在。
  2. 工业资本主义：**18世纪至19世纪，资本主义主要以工业生产的形式存在。
  3. 金融资本主义：**19世纪末至20世纪初，资本主义主要以金融业为主导。
  4. 全球化资本主义：**20世纪末至今，资本主义以全球化的形式存在。

资本主义对现代社会的影响

吉登斯认为，资本主义的发展对现代社会产生了深刻的影响。这些影响包括：

  1. 经济增长：**资本主义经济的快速增长带来了物质财富的极大丰富。
  2. 社会分化：**资本主义经济的发展导致了社会阶层的分化，出现了资产阶级和无产阶级。
  3. 城市化：**资本主义经济的发展导致了城市人口的快速增长，出现了大城市和城市化地区。
  4. 全球化：**资本主义经济的发展导致了全球经济的相互联系，形成了全球化的世界经济。
  5. 现代性：**资本主义经济的发展带来了现代性的价值观和生活方式，如个人主义、理性主义和科学主义。

吉登斯的结构化理论

吉登斯提出了“结构化理论”来解释资本主义的发展及其对现代社会的影响。结构化理论认为，社会结构既是行动的产物，又是行动的条件。社会结构为行动提供了框架和限制，但行动者也可以通过他们的行动来改变社会结构。

吉登斯认为，资本主义经济是一种结构，它既是行动的产物，又是行动的条件。资本主义经济为人们的经济活动提供了框架和限制，但人们也可以通过他们的经济活动来改变资本主义经济。

  1. 资本主义的经济增长：**资本主义经济的快速增长带来了物质财富的极大丰富。这使得人们的生活水平得到了显著提高，也为社会的发展提供了物质基础。
  2. 资本主义的社会分化：**资本主义经济的发展导致了社会阶层的分化，出现了资产阶级和无产阶级。这导致了社会的矛盾和冲突，也为社会变革提供了动力。
  3. 资本主义的城市化：**资本主义经济的发展导致了城市人口的快速增长，出现了大城市和城市化地区。这带来了许多社会问题，如住房问题、交通问题和环境问题。
  4. 资本主义的全球化：**资本主义经济的发展导致了全球经济的相互联系，形成了全球化的世界经济。这使得世界各国之间的经济联系日益紧密，也为全球合作与发展提供了机遇。

吉登斯的结构化理论与马克思的唯物史观有相似之处，但也有不同之处。马克思认为，经济基础决定上层建筑，经济结构决定了社会结构。吉登斯则认为，社会结构既是行动的产物，又是行动的条件。社会结构为行动提供了框架和限制，但行动者也可以通过他们的行动来改变社会结构。

总结

在英国，自从进入20世纪，以马林诺夫斯基为代表的文化人类学派反省意识后，近半个世纪，英国再没有与韦伯、帕森斯、戈夫曼等比肩的大社会学家。直到70年代，年仅33岁的吉登斯，以《资本主义与现代社会理论》横空出世，社会学又出了一个巨星级人物。这本书的副标题是《对马克思、涂尔干和韦伯著作的分析》，一看就令人激动的标题，也说明了吉登斯的功力非同一般。随后，吉登斯出版了一系列著作，累计30多部。他在社会学上的成就，导致一直被人只看成一位社会学家，而忽略他是一位政治哲学家。

很多朋友在学习政治哲学时，忘记了这样一种路径：政治哲学可以从哲学入手，也可以从经济学、社会学、政治学、法学等入手。比如 哈耶克是从政治经济学走进政治哲学领域的；吉登斯是从社会学走进政治哲学的。

他从社会学角度分析了现代性，标志性著作就是《现代性的后果》。他分别把20世纪最棘手的现代性危机问题做了全面分析和总结。他在《现代性的后果》开篇第一句就是：在本书中，我将按文化与认识论的比较，对现代性做出一种制度性分析。

他指出，现代的社会制度在组织方面是独一无二的，其在形式上优于所有类型的传统秩序。他认为过去的社会的类型已经变化，而对新的类型又缺乏制度性分析，而马克思、涂尔干、韦伯等对社会的分析，尤其解释现代性问题上，都倾向某种单一的驾驭社会巨变的动力。马克思聚焦资本主义，涂尔干则侧重工业化，韦伯关注官僚社会的客层组织。

他提出，我们面临着严峻的现代性问题，比如剧烈增长、经济机制崩溃、生态危机、核冲突与大规模战争，提出人类走出这种困境的思路。吉登斯的社会学理论很丰富，在现代性危机问题上，比哈贝马斯等人更具说服力。

他在《第三条道路——社会民主主义复兴》里，提出自己的政治哲学思想，即《第三条道路》，就是指既不做哈耶克式的老右派的自由主义，也不做马克思式的老左派的社会民主主义，而是积极的社会民主主义。

他提出平等、保护弱者、有限基因管理，复兴公共空间、积极福利、社会投资性国家，超越劳动的社会等政治哲学观点。

他非常关注西方社会两极分化而严重的问题，可以说，吉登斯的《第三条道路》本质上还是社会路线，市民路线，现实路线。他深受马克思关于无产阶级思想影响。

吉登斯也毫不掩饰自己深受马克思主义的影响，在人、社会、国家三者关系里，吉登斯把人和社会关系放在人和国家关系之前，提出不承担责任就没有权利。他这句话同时指民众，也是政府，意思是，如果政府不为人和社会承担什么，它就失去公权力，民众不积极参与和改善自己，也不受平等庇护。

吉登斯提到，社会吸纳变成单一吸纳劳动，社会和人的关系仅仅发生在劳动关系上，这是不可取的。

总而言之，吉登斯重点关注人和社会这个关系组。他借马克思的话说道，在马克思看来，随着充分发达的社会主义社会的到来，国家将会消亡，在这种社会中，每个人的自由发展将是所有自由发展的条件。吉登斯深感，目前人不仅被国家束缚，也被社会冷漠对待，人与人又彼此失去传统关系，这种现代性危机带来后现代的生存状况，导致社会共同价值和个体自由空洞化、理论化。

吉登斯的政治哲学思想，有马克思的社会主义思想，也有桑德尔的共同主义，也有阿伦特的个人政治自由和公共名誉，同时他也批判新古典自由主义，指出自由主义带来的现代化的同时，也带来现代性困境，社会撕裂。

吉登斯被称为20世纪思想的巨人。吉登斯不仅是英国政治经济学院院长，终身教授，也是英国布莱尔时期的政府顾问。他在学术上和现实政治中，同样很有影响力。他对同时代哈贝马斯的一些理论，也提出过批评。
