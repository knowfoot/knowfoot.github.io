爱因斯坦和波尔之间一场著名的论战

由于对世界本性的不同认知，爱因斯坦和哥本哈根学派之间争论不断。这引发了爱因斯坦和波尔之间一场著名的论战。

第五届索尔维会议-量子力学

1927年，比利时国王赞助召开了第五届索尔维会议，会议持续六天，主题正式量子力学。

在会议前期，劳伦斯·布拉格和康普顿分别对 X 射线和康普顿散射实验做了报告。随后，量子理论的交锋正式开始。

德布罗意首先在报告中介绍了他的德布罗意波，并提出了一种导波理论，他将波函数视为引导粒子动作的一种向导波。但由于缺乏因果的逻辑一致性，德布罗意的理论遭到了泡利的猛烈批评。最后，德布罗意顶不住炮火，放弃了自己的观点。

紧接着，薛定谔上台介绍了他的波动力学。但对于他的薛定谔方程，他仍坚持那个传统的解释——薛定谔认为波函数或许是一个物理实在的可观测量，表示的是电子的电荷在空间中的实际分布。对于这个观点，不仅哥本哈根学派表示反对，连自己人爱因斯坦都表示无法接受。

泡利则直言薛定谔的观点幼稚，海森堡则说“我从薛定谔的计算中看不到任何东西可以证明事实如同他所希望的那样。”薛定谔也承认他的计算确实还不太严谨，但他依然坚持认为“所谓的电子的轨道就是胡扯。”波恩则回敬道，波函数和波函数的分布一点都不是胡扯。

海森堡和波恩认为波函数是个不可观测量，波函数的平方代表粒子在空间某处出现的概率，电子本身并不会像波那般展开，只是它出现的概率确实像一个波，严格地按照波函数的分布展开。如此一来，量子规律本质上是统计性的而非决定论的。

在波恩和海森堡的猛烈轰炸下，薛定谔最终也只得投降。

最后是波尔演说。演说中他再次阐述了他互补性原理的思想，主要是说物质的波动性和粒子性就像是硬币的两面，两者的不可分割。它具体如何展现取决于实际观测的手段。

在整场报告中，作为绝对 C 位的爱因斯坦却始终保持沉默，很少表态。即使发言也非常的谦虚，比如“抱歉，我对量子力学研究的不太深，就简单地说点我个人的看法吧。”

而等到报告结束后，进入自由讨论环节时，爱因斯坦出手了。爱因斯坦的招数是他最擅长的思想实验。

据当事的海森堡回忆，当时整个自由讨论环节，很快就变成了一场爱因斯坦和波尔之间的单挑。通常是在旅店一起吃早餐时，爱因斯坦就提出来一个思想实验，说“如果先这样再这样，最后再这样，那量子力学就有问题，请问你波尔该如何应对？”

一到中午，波尔就赶紧召集海森堡和泡利一起研究，然后晚餐时，波尔就破解了爱因斯坦的招数。然后晚上爱因斯坦又想出一个思想实验，第二天一大早继续甩给波尔。

如此进行了好几个回合，在这几轮交锋中，有一个最为著名，它就是单缝实验。

假设有一个带单缝的遮光板，遮光板被一个超高精度的弹簧挂起来。当一个电子从缝中垂直穿过后，它就会在上下方向上发生衍射，这时根据动量守恒定律，遮光板就会获得一个往下的动量，弹簧就应该往下伸展一点。相反，如果电子往下，弹簧就应该往上。

爱因斯坦说，根据弹簧的收缩情况，我们就能反推出电子通过单缝时的动量。同时我也知道单缝的位置，那我就同时知道了电子的位置和动量，这不就违反了海森堡不确定性原理了吗？

波尔乍一听到有些懵，但经过仔细讨论和思考，波尔还是发现了问题所在，反驳道：“既然电子这种微观粒子都能让弹簧发生一次震动，那遮光板和弹簧必须也是量子系统。而既然是量子系统，细缝本身的位置和动量就具有不确定性，所以也不能根据细缝的位置和动量来反推电子的状态。这里爱因斯坦混淆了宏观世界和量子世界。”

爱因斯坦对波尔的说法无法反驳。

第五届索尔维会议，哥本哈根学派轻松以 3 比 0 结束论战，爱因斯坦也逐渐意识到量子力学不是那么容易被推翻的。哥本哈根学派从这一刻起在物理学界站稳了脚跟。

第六届索尔维会议-光盒

但显然，爱因斯坦打从心底还是不服气的。三年后的第六届索尔维会议，爱因斯坦有备而来，一上台就给波尔甩了一个思想实验。

爱因斯坦在黑板上画了一个盒子，假设盒子里面装满了光子，我们称之为光合，光合里有个时钟，光合上有可以开关的小孔。我们选定一个时间点，开启小孔一段时间，让一个光子飞出，然后称一称箱子的重量。根据爱因斯坦质能方程，虽然光子的静止质量为零，但光子拥有能量，这等效于光子拥有一定的动质量。当光子离开光合时，光合的重量就会发生一个小小的改变。而我们又知道光子飞出的精确定时间，这不就违反了海森堡所说的能量和时间之间的不确定性原理吗？

注意，在这样的实验中，不需要在实验精度上纠缠，毕竟这只是个思想实验。而只需要从原理上找到漏洞即可。

听完爱因斯坦的这个实验后，波尔当常大惊失色。

我永远也忘不了，这两个对手离开会场时的情景，爱因斯坦的身影高大威严，嘴角挂着略带一丝嘲讽的笑容大踏步前进。波尔则是跟在后面一路小跑，在一旁激动的对爱因斯坦不停念叨着：“这不对啊，你这个实验要真是对的，那物理学就完蛋了。”

当天回去后，波尔茶饭不思，一直想到凌晨时分，终于恍然大悟，爱因斯坦犯了一个巨大的错误。

第二天一大早，波尔向爱因斯坦发起了反击。

波尔说，要称光合的重量你总得使用仪器。我们可以假设光合是悬挂在无限精密的弹簧秤上，将控制小孔开合的快门与箱内的时钟相连。当光子飞出时，弹簧会向上收缩一下，弹簧的高度代表光合的重量。

但是，弹簧的高度有一个不确定性，这代表了光合重量的不确定性。另一方面，根据你的广义相对论，越高的地方时间就过得越快，对吧？所以高度的不确定性也代表了光合时间的不确定性。经过计算，刚好满足海森堡的不确定性原理。

在最后的交锋中，爱因斯坦混淆宏观世界和量子世界，忘记了自己的广义相对论。爱因斯坦这次彻底破防了。

经过这两次与波尔的论战，爱因斯坦不得不承认量子力学确实有点东西的。

EPR佯谬

其实自始自终，两人的辩论都非常友好。爱因斯坦也并非你想象中那样，是一个顽固的老头。开玩笑，那可是敢于颠覆千百年来的绝对时空观，敢于废除万有引力的男人。爱因斯坦完全不否认哥本哈根学派带来的贡献。一年之后，爱因斯坦还特地向诺贝尔奖委员会推荐了薛定谔和海森堡，让他们拿到了诺贝尔物理学奖。

此后，欧洲政治局势每况愈下。1933年，爱因斯坦移居美国，加入了普林斯顿高等研究院。未能参加第七届索尔维会议，失去了大佬作证，德布罗意和薛定谔自然也就掀不起什么风浪。在第七届索尔维会议上，他俩基本就是静静地看着哥本哈根学派表演。

虽然爱因斯坦已经逐渐脱离了主流的物理圈，但他与玻尔的较量仍在继续。他承认量子力学在逻辑上是自洽的，但他总觉得这里面还是有点不太对劲。就比如，哥本哈根解释认为，粒子在我们不观测时，是处于一种模糊的量子叠加态。而我们一观测，就瞬间随机地坍塌为某一个确定的状态。这和爱因斯坦的世界观严重不符。爱因斯坦认为世界应该是客观实在的，不受主观影响的。你之所以觉得这是纯随机的，也许是量子力学本身还不够完备，这里面也许还有其他的什么机制。

1935年，爱因斯坦发起了《最后一集》。他和两个同事合写了一篇论文，标题是《描述物理实在的量子力学是完备的吗？》。论文中又提出了一个思想实验，后人将该实验以三人的手字母命名，称为EPR佯谬。EPR佯谬不但在物理学上，而且在哲学上都达到了一个前所未有的高度。

简单来说，假设有一个大粒子，因为衰变等原因分裂成两个全同的小粒子，然后沿直线各自往相反方向分离。根据动量守恒，两个粒子的动量时刻保持相反，所以它俩的位置也必定保持对称。现在我先测量一下A的位置为X，也就相当于知道了B的位置为负的X。根据海森堡不确定性原理，测量A的位置会破坏A的动量，但这对B没有影响，对吧？然后我再测量一下粒子B的动量，于是我就同时获得了B的位置和动量，请问你海森堡如何解释？

EPR佯谬已经提出，哥本哈根学派立即大乱阵脚。玻尔也立刻放下手头所有工作全力应战。经过一番讨论，玻尔给出的反驳是这样的。首先由于AB两个粒子是由同一个大粒子分裂而来的，所以它俩应该被看作是由同一个波函数描述的一个整体。此时你测量A的位置就等同于测量了B的位置，B的动量也就被影响了。此时你再去测量B的动量，那就已经不是之前的动量了，所以你还是不能同时获得粒子的位置和动量。

但这一回爱因斯坦不买账了。好家伙，我这两个粒子可以相隔几光年之远。如果测量A的状态立刻就能破坏B的状态，那这不就成了一种鬼魅般的超距作用了吗？后来薛定谔为了嘲讽玻尔，说你的意思是这俩粒子难不成还能纠缠在一起？这一回爱因斯坦没有犯任何错误。爱因斯坦成功的论证了，你量子力学要想是对的，那这种鬼魅般的超距作用就必定存在。而这一点连玻尔自己都表示难以接受。

这次论战将这两个对手世界观的分歧展露得淋漓尽致。爱因斯坦始终认为，世界应该是实在性的，是确定性的，也就是世界应该是独立于人类观察者的客观实在，并且不存在任何超光速的瞬时相互影响。而玻尔则认为，你所谓的客观实在与观察的动作有关，在你观测前，压根就不存在所谓的客观实在。观测动作本身就是测量结果的一部分。

总的来说，这次玻尔和爱因斯坦谁也没说服谁。这场争论也并没有继续下去。在此后的长达30年时间里，那个鬼魅般的超距作用一直是个谜。爱因斯坦的质疑也逐渐被世人所遗忘。直到后来贝尔不等式的出现，以及量子纠缠实验的成功，问题才得到解决。

不过可惜的是，爱因斯坦和玻尔都未能亲眼见到这些实验结果。在生命的最后30年里，爱因斯坦一直都在寻找着一种、比量子力学更基本的理论，但受限于实验技术水平，这注定是一条无比艰难的路。1955年，爱因斯坦因主动脉破裂，孤零零地去世了。不久之后的1962年，玻尔也因心脏衰竭去世。就在玻尔去世前一天还在用的黑板上，人们发现了一个图形，那正是爱因斯坦的光盒。
